# LSU Beamer Theme

A LaTeX Beamer presentation template designed for Louisiana State University (LSU).

<!-- Add a screenshot here -->
![Preview](preview.png)

## Usage
```latex
\documentclass{beamer}
\usetheme{LSU}

\title{Your Presentation Title}
\author{Your Name}
\institute{Louisiana State University}
\date{\today}

\begin{document}

\begin{frame}
    \titlepage
\end{frame}

% Your slides here

\end{document}
```

## Installation

1. Clone this repository:
```bash
   git clone https://github.com/yourusername/lsu-beamer-theme.git
```
2. Copy the theme files to your project directory or your local texmf tree.

## Files

- `beamerthemeLSU.sty` ‚Äî Main theme file
- `beamercolorthemeLSU.sty` ‚Äî Color definitions (LSU purple & gold)
